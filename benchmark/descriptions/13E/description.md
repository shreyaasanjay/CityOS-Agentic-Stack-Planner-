# Pharmaceutical Lab Synthesis

Three scientists collaborate on synthesizing and testing a new pharmaceutical compound. They share two expensive instruments that only one scientist can use at a time.

## Agents
- **CHEMIST**: synthesizes and purifies the compound
- **BIOLOGIST**: tests biological activity of the compound
- **ANALYST**: runs quality analytics and generates certification

## Shared Resources
- **SPECTROMETER**: molecular analysis instrument (one scientist at a time)
- **CENTRIFUGE**: separation/purification instrument (one scientist at a time)

## Workflow

### CHEMIST
1. Use the **SPECTROMETER** to analyze raw materials
2. Use the **CENTRIFUGE** to purify the compound
3. Send the purified compound sample to the **BIOLOGIST** for testing
4. Wait for the quality report from the **ANALYST**
5. Use the **SPECTROMETER** again to run final compound validation
6. Done

### BIOLOGIST
1. Use the **CENTRIFUGE** to prepare cell cultures
2. Use the **SPECTROMETER** to calibrate fluorescence measurements
3. Receive the compound sample from the **CHEMIST**
4. Run bioassay — test compound on cell cultures (no instrument needed)
5. Send test results to the **ANALYST**
6. Done

### ANALYST
1. Receive test results from the **BIOLOGIST**
2. Use the **SPECTROMETER** for detailed molecular analysis
3. Use the **CENTRIFUGE** for additional separation verification
4. Send the quality report to the **CHEMIST**
5. Done

## Goal
All three scientists complete their work: the **CHEMIST** validates the compound, the **BIOLOGIST** completes the bioassay, and the **ANALYST** delivers the quality report.
