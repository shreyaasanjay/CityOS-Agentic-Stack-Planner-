# Research Paper Writing — 7 Agents

Seven researchers are co-authoring a paper with 7 sections (intro, background, related work, methods, experiments, results, discussion). Each section can only be edited by one researcher at a time.

## Agents
- **RESEARCHER_A**: Writes the connection between introduction and background. Needs to edit **INTRO_SECTION** and **BACKGROUND_SECTION**.
- **RESEARCHER_B**: Writes the connection between background and related work. Needs to edit **BACKGROUND_SECTION** and **RELATED_SECTION**.
- **RESEARCHER_C**: Writes the connection between related work and methods. Needs to edit **RELATED_SECTION** and **METHODS_SECTION**.
- **RESEARCHER_D**: Writes the connection between methods and experiments. Needs to edit **METHODS_SECTION** and **EXPERIMENTS_SECTION**.
- **RESEARCHER_E**: Writes the connection between experiments and results. Needs to edit **EXPERIMENTS_SECTION** and **RESULTS_SECTION**.
- **RESEARCHER_F**: Writes the connection between results and discussion. Needs to edit **RESULTS_SECTION** and **DISCUSSION_SECTION**.
- **RESEARCHER_G**: Writes the connection between discussion and introduction. Needs to edit **DISCUSSION_SECTION** and **INTRO_SECTION**.

## Shared Resources
- **INTRO_SECTION**: Introduction section of the shared document
- **BACKGROUND_SECTION**: Background section of the shared document
- **RELATED_SECTION**: Related work section of the shared document
- **METHODS_SECTION**: Methods section of the shared document
- **EXPERIMENTS_SECTION**: Experiments section of the shared document
- **RESULTS_SECTION**: Results section of the shared document
- **DISCUSSION_SECTION**: Discussion section of the shared document

## Workflow

Each researcher first reads literature and drafts content locally. When ready to write into the shared document, they write their content into the required sections, then review their own work locally — if the review finds issues, the researcher reports the finding and finishes.

## Goal
All seven researchers complete their literature review, drafting, writing, and review phases without conflicts over shared document sections.
