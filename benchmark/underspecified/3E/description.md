We have two researchers and an editor putting together a report. Each researcher digs into their own subtopic first, then writes their section into a single shared draft and adds their citations to a common reference list. Only one person can write in the draft at a time, and likewise only one can update the reference list at a time — otherwise edits get clobbered. When a researcher finishes their section, they hand it off to the editor and let them know it's ready. The editor shouldn't start reviewing until both sections are in. During review, the editor judges whether the two sections are consistent with each other; if one isn't, the editor sends it back to that researcher, who revises it in the shared draft and resubmits, and that can repeat. Once both sections pass, the editor merges them into the final report — that merging happens in the same shared draft, so the editor needs solo access then too. We're done when the combined report exists and nobody has stepped on anyone else's edits.
